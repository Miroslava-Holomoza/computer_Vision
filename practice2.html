Одиниці вимірювання в HTML та CSS

Вступ

Одиниці вимірювання в HTML та CSS використовуються для визначення розмірів тексту, зображень та інших елементів вебсторінки. Від правильного вибору одиниці залежить читабельність матеріалу та загальний вигляд сторінки.

Абсолютні одиниці

Абсолютні одиниці мають фіксоване значення і не залежать від інших елементів сторінки. Найпоширенішою одиницею є піксель (px).

Відносні одиниці

Відносні одиниці змінюються залежно від розміру шрифту або параметрів вікна браузера. Наприклад, em, rem, %, vw, vh.

Де застосовуються одиниці вимірювання

  • розмір тексту
  • ширина зображень
  • міжрядковий інтервал
  • відступи

Корисні ресурси

Більш детальну інформацію можна знайти на офіційному ресурсі W3C або в довіднику з CSS. Для прикладу можна скористатися енциклопедією Вікіпедії.

Цитата

Правильний вибір одиниць вимірювання робить вебсторінку адаптивною та зручною для користувачів.

Вибір правильної одиниці — ключ до якісного дизайну.
